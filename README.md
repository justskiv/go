# April Fools' Prank: Go Inheritance Proposal

A pixel-perfect replica of a GitHub issue page for the Go repository, proposing to add classical inheritance to Go.

Built as an April 1st prank for the Go community.

## What is this?

A static HTML page that looks exactly like a real GitHub issue in [golang/go](https://github.com/golang/go):

- Dark theme matching GitHub's current design, desktop and mobile
- Detailed proposal with syntax-highlighted Go code examples
- Comments from community members and Go team leads moving the proposal to **Likely Accept**
- Working UI elements — dropdowns, copy buttons, search
- Real GitHub avatars and profile links
- Open Graph tags for convincing link previews in Telegram and social media

## Easter Egg

After some time on the page, things start to go wrong:

- **~4s** — subtle screen flickers
- **~8s** — text starts corrupting, one character at a time
- **~12s** — corruption accelerates, screen starts shaking
- **~18s** — chaos: rapid corruption, color shifts
- **~24s** — black screen, Go runtime panic typed out from `proposal/inheritance.go`
- **~28s** — audio kicks in

Timings are configurable in the `T` array in the script.
